Brief Project Description:
This project is a Profile Management System developed using HTML, JSP, Servlet, JavaBean, JDBC, and Java DB. The system allows users to save and manage student profile information in a database.

Implemented Features:
1. Add new student profile
2. View all profiles
3. Search profile by Student ID or Name
4. Delete profile
5. Form validation
6. Database integration using JDBC
